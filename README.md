# PropNex Malaysia Newsletter

A weekly newsletter archive for PropNex Malaysia agents. Every Friday, the week's PropNex news in one place.

## Site

- `index.html` — the archive landing page (lists all issues)
- `PropNex-Newsletter-Issue-01.html` — Issue 01

Both pages are fully self-contained (images are embedded), so they render anywhere with no external assets.

## Hosting

This site is published with GitHub Pages. The live archive is available at the Pages URL for this repository.
